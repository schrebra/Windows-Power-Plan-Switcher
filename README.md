# Windows Power Plan Switcher

## What is this?
The **Power Plan Switcher** is a simple, lightweight tool designed to make changing your computer's power settings fast and easy. Instead of clicking through several Windows menus, this tool gives you a clean, small window and a tray icon to swap between "Power Saver," "Balanced," and "High Performance" modes with a single click.

## What it does
This tool acts as a shortcut for Windows' built-in power management settings. It monitors your computer to see which power plan is currently active and lets you instantly switch to a different one. It also features a "system tray" mode, allowing you to hide the main window while keeping a small, color-coded icon in your taskbar to show you exactly which mode you are using at a glance.

---

## Who is it for?
* **Laptop Users:** Easily switch to "Power Saver" mode when you're low on battery, or "High Performance" when you're plugged in and need extra speed.
* **Anyone who wants simplicity:** If you find the Windows settings menu tedious to navigate, this provides a much faster, one-click solution.
* **Users who like a clean look:** It uses a modern, dark-themed design that blends well with any desktop setup.

## Features
* **One-Click Switching:** Change your power plan instantly from the app window or the taskbar.
* **Status Updates:** The app automatically shows you which plan is active with color-coded highlights (Green for Saver, Blue for Balanced, and Red for High Performance).
* **System Tray Integration:** Minimize the app to your taskbar (near the clock) so it doesn't clutter your screen. You can right-click the icon to change settings or close the app entirely.
* **Modern Design:** Features a polished, dark aesthetic that feels like a native part of your system.
* **Automatic Detection:** It automatically updates its own icon and status whenever you change your power plan.

## How to get started
1. Copy the code into a plain text file.
2. Save the file with a `.ps1` extension (e.g., `PowerSwitcher.ps1`).
3. Right-click the file and select **"Run with PowerShell."**
4. The window will appear, allowing you to select your preferred power mode immediately.
